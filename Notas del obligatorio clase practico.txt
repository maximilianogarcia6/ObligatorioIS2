tuvimos que hacer un proyecto vacio en netbeans y copiar la carpeta src para poder abrirlo y ejecutarlo en dicha herramienta
En eclipse abre de forma nativa

Se debe utilizar al menos tres herramientas y compararlas entre sí.

Tiempo de instalación de herramientas - 2 horas

El paquete de test arrojó problemas al faltar la libreria Junit. Se realizó la importación de la misma

Usabilidad: en pcs con altas resoluciones (4k) la interfaz se ve muy chica y apenas se distinguen las letras.

Usaremos GIT como repositorio ya que es distribuido y gratis, ademas de ser la herramienta estudiada en cursos pasados. Para registro de incidentes usamos github ya que viene integrada al manejo del repositorio. Descartamos otras herramientas como mantis, redmine o bugzilla ya que requiere un tiempo de instalación y configuración al tener que montar un servidor web, además de tener estas ultimas la desventaja de ser para trabajo local, frente a github que está en internet.
